Installation

Prerequisites

  • Frappe Framework v13 or higher
  • Python 3.7+
  • WhatsApp Business API account

Step 1: Get the App

bench get-app https://github.com/shridarpatil/frappe_whatsapp.git

Step 2: Install on Site

bench --site your-site.com install-app frappe_whatsapp

Step 3: Migrate

bench --site your-site.com migrate

Step 4: Restart Bench

bench restart

Configuration

1. WhatsApp Business API Setup

  1. Create a WhatsApp Business account
  2. Get your API credentials from Meta Business Platform
  3. Note down your Phone Number ID and Access Token

2. Configure in Frappe

  1. Navigate to WhatsApp Settings in your Frappe site
  2. Enter your WhatsApp Business API credentials
  3. Configure webhook URL for receiving messages
  4. Set up message templates as needed

3. Webhook Configuration

Set your webhook URL in Meta Business Platform:

https://your-site.com/api/method/frappe_whatsapp.webhook

4. Verify Setup

  1. Send a test message from WhatsApp Settings
  2. Check message logs for delivery status
  3. Test incoming message reception

API Documentation

Send Message

POST /api/method/frappe_whatsapp.send_message

Parameters:

  • to: Recipient phone number (with country code)
  • message: Message content
  • template: Template name (optional)

Get Message Status

GET /api/method/frappe_whatsapp.get_message_status

Parameters:

  • message_id: WhatsApp message ID

Webhook for Incoming Messages

POST /api/method/frappe_whatsapp.webhook

Automatically handles incoming messages and status updates from WhatsApp

Python Usage Example

import frappe
from frappe_whatsapp.api import send_whatsapp_message

# Send a simple text message
send_whatsapp_message(
    to="+1234567890",
    message="Hello from Frappe WhatsApp!"
)

# Send using a template
send_whatsapp_message(
    to="+1234567890",
    template="order_confirmation",
    parameters=["ORD-12345", "John Doe"]
)
